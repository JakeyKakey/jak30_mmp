Contents:

/bin/

The bin folder contains the Main.py file, and the database, tax and xmlsamples folders.

/bin/database/

The database folder contains the taxonomy folder containing names-trimmed.txt and rinkedlineage.dmp file used as part of the program.

/bin/xmlsamples

The xmlsamples folder contains 12000, 1000, and 100 result BLAST XML samples, as well as an empty XML file used for test purposes. 

/bin/tax

The tax folder contains the TaxTree.py and TaxMatch.py and their associated test classes. 

/bin/tax/testdata

The testdata folder contains some serialized files used as part of automated testing.
